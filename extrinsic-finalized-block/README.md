# SubQuery Example - Extrinsics within Finalized Blocks

This subquery will index extrinsics and so they can be queried by hash

# Get Started
### 1. install dependencies
```shell
yarn
```

### 2. generate types
```shell
yarn codegen
```

### 3. build
```shell
yarn build
```

### 4. run locally
```shell
yarn start:docker
```
